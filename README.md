# a1010841065.github.io
个人博客（ http://www.4dog.win ）的静态备份

### 运行环境:

[Git](https://git-scm.com/downloads)

[Node.js](https://nodejs.org)

### 使用方法:

```
#克隆源文件
git clone https://github.com/a1010841065/a1010841065.github.io.git

#进入源目录
cd a1010841065.github.io

#配置Hexo环境
npm install

#撰写文章
hexo new "文章标题"

#提交博客更改
hexo clean
hexo g -d

#提交源文件
git add .
git commit -m "更新内容"
git push
```
